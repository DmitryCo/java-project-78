### Description
A data validator is a library that can be used to check the correctness of any data.

### Setup:
```bash
    git clone git@github.com:DmitryCo/java-project-78.git
    cd java-project-78
    make build
```

### Install:
```bash
    make install   
```

## Run-dist
```bash
    make run-dist
```

## Run
```bash
    make run
```

## Run test
```bash
    make test
```

## Run report
```bash
    make report
```

## Run checkstyle
```bash
    make lint
```
